# 质量奖励模型数据与失败证据

## 成对历史评审 RM

训练/校准/测试分别为 597/130/130 条，另含 30 条对抗样本和 130 条冻结测试预测。正式测试 Spearman 为 0.339、MAE 为 1.800、pairwise accuracy 为 0.474；风险头高召回但精度很低，因此未通过质量奖励模型验证。

## 绝对评分教师数据

完整训练集共 5760 条；公开仓库只保留前 500 条样本，并完整保留 720 条校准集与 720 条测试集。完整版训练集留在私有归档。该路线完成了教师标签和校准训练，但没有形成冻结测试集上的终局判决，因此只能作为未完成实验资产，不能宣称模型通过或失败。

汇总分析见 `results/calibration.md` 与 `results/metrics/quality_rm_failure.json`。
